(1)应用程序功能的详细说明；本程序为一个休闲娱乐游戏大鱼吃小鱼，首先进行用户名密码的注册与登录，若已注册则可直接登录，登陆成功后进入有玩法说明，第一关，第二关，退出游戏等交互按钮的菜单界面，鼠标点击对应按钮进入相应界面，建议先点击“玩法说明”按钮查看规则，再点击“返回菜单”按钮回到菜单界面选择关卡进入游戏，注意需先通过第一关才可解锁第二关，游戏过程中播放背景音乐，玩家通过键盘按键上下左右↑↓←→操控角色移动，游戏会随机生成自动移动的各种不同鱼类，角色鱼每吃掉一条鱼积分+10，注意若碰到比自己大的鱼则游戏失败返回菜单界面，本游戏采用积分制，积分累积到100（即成功吃掉10条）则游戏胜利，返回菜单界面，若需结束游戏则点击菜单界面“退出游戏”按钮关闭界面

(2)  应用程序运行环境的要求；Visual Studio 2022，EasyX库，C语言

(3)  应用程序的安装与启动方法；打开程序文件并运行

(4)  程序的界面、交互方式和操作方法；菜单界面点击矩形按钮跳转至相应界面，通过键盘按键上下左右↑↓←→操控角色移动

(5)  输入数据类型、格式和内容限制；按要求输入fillout或login（注意均为小写，并注意拼写正确），用户名不超过20个字符，密码不超过8个字符

(6)  在应用程序运行中，用户需要使用的交互命令名称、功能和格式的详细说明。

在注册登录界面，若需注册，则输入fillout后回车输入用户名，回车输入密码，再次回车，三个操作完成即为注册成功；注册成功后再次输入用户名回车，输入密码回车完成登录；若已注册需直接登录，则输入login回车，输入用户名回车，输入密码回车，三个操作完成后即为登录成功，随即进入菜单界面。在菜单界面鼠标点击相应按钮进入对应界面。注意需先通过第一关才可解锁第二关。游戏时用键盘上下左右操控角色移动，游戏失败或胜利后均有提示弹窗，鼠标点击弹窗确定按钮回到菜单界面。

1.用户名密码注册登录

2.有玩法说明，第一关，第二关，返回菜单等交互按钮的菜单界面

3.鼠标点击对应按钮进入相应界面

4.游戏过程中播放背景音乐

5.随机生成左右朝向的自动移动的各种不同鱼类

6.键盘按键上下左右↑↓←→操控角色移动

7.每吃掉一条鱼积分+10

8.判定鱼的大小关系与是否吃掉此鱼，成功吃掉则积分+10，碰到比自己大的则游戏失败，积分累积到100（即成功吃掉10条）则游戏胜利

9.游戏失败的弹窗与游戏胜利的弹窗，并返回菜单界面
